führen Sie folgende Schritte aus, um die Webapplikation nutzen zu können:
1. Entpacken Sie die ZIP-Datei in Ihrem htdocs Ordner
2. Bitte besuchen Sie die URL http://localhost/phpmyadmin/server_import.php und importieren Sie dort im Abschnitt "Zu importierende Datei:" die Datei htdocs\Sales Power\database_script.sql, damit die Datenbank angelegt wird.
3. Hinterlegen Sie Ihre Anmeldedaten für die Datenbank in der Datei htdocs/Sales Power/util/database_manager.php
4. Besuchen Sie anschließend "http://localhost/Sales%20Power/index.php" und melden Sie sich mit einem vordefinierten Account an:
Benutzername: root
Passwort: root
